# 끝말잇기 Pro (Word-Chain Pro)
AI vs 사람 / 사람 vs 사람 — 점수/타이머/패스/힌트/대용량 사전.

## 기능
- 난이도(쉬움/보통/어려움)
- 점수제: 유효 단어 1점, 목표 점수 먼저 달성 시 승리
- 턴 제한(0=무제한), 시간초과 자동 패스
- 패스 횟수 설정
- 힌트(가능 단어 수)
- 옵션: 고유명사 허용, 최소 글자수, 금지 끝 글자(쉼표로 입력)
- 사전: 파일 업로드 / URL 로드 / localStorage 저장
